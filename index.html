Iris uses CSS3 Gradients for everything, so it'll look awesome on HiDPI displays out of the box. Initially developed for Custom Colors on WordPress.com

Documentation

Found on the project page.

Example: (This title will change color)

The code for the above is simple:

jQuery(document).ready(function($) {
  $("#color").iris({
    hide: false,
    change: function( event, ui ) {
      $("#example").css( "color", ui.color.toString() );
    }
  });
});
